ЧТО ТАКОЕ ШКОЛА БЕЗОПАСНОСТИ?

Этот портал предназначен для обучения пользователей и предупреждения мошенничества в интернете.

Наш обучающий курс предоставляет актуальную информацию в форме теста.

Он состоит из 20 вопросов, которые покрывают 4 основных блока теории.

Наш курс разработан таким образом, чтобы учащиеся могли протестировать и углубить свои знания в каждом из этих блоков.

Наша цель - предоставить полноценный обучающий опыт, который поможет учащимся освоить основы данной темы и улучшить свои знания.

Каждый вопрос имеет несколько вариантов ответов, из которых учащиеся выбирают правильный.

По результатам теста, пользователю на электронную почту будет выслан сертификат, подтверждающий прохождение курса.
